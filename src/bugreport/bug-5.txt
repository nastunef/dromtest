Title: При сложении округление дробных чисел до двух знаков после запятой
Description: После операции сложения двух чисел происходит округление
    первого слагаемого, второго слагаемого и результата до двух знаков после запятой
Precondition:
    Платформа Android 5.1.
    Приложение AndroidSimpleApp version 1.0
Steps To Reproduce:
    1. Ввести в левое поле для ввода первого числа (inputFieldLeft) значение 1.1
    2. Ввести в правое поле для ввода второго числа (inputFieldRight) значение 12.4444444
    3. Нажать +

Expected Result:
   1.1 + 12.4444444 = 13.5444444
Actual Result:
   1.10 + 12.44 = 13.54
